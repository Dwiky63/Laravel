## 🔧 Common Customizations

### Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  accent: '#E63946',  // Change to your color
}
```

### Update Content
Edit component files in `resources/js/components/`:
- Navbar: Navigation links
- Hero: Main headline
- About: Company values
- Products: Product information
- News: Article information
- Events: Event details
- Footer: Links and contact

### Modify Animations
Edit `tailwind.config.js` keyframes or `app.css`

### Change Fonts
Edit Google Fonts link in `landing.blade.php`

### Add More Sections
1. Create new component file
2. Import in `App.jsx`
3. Style with Tailwind CSS
